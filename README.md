# 📶 WiFi Monitor System

## 📌 Overview
The WiFi Monitor System is a desktop-based application designed to monitor network performance in real-time. It tracks key metrics such as connection status, latency, signal strength, and overall network quality.

This project was developed to provide users with a simple and efficient way to diagnose unstable internet connections without relying on browser-based tools.

## 🎯 Objectives
- Monitor WiFi connection in real-time
- Measure network latency and stability
- Display network performance metrics
- Provide a simple and user-friendly dashboard

## 🚀 Features
- ✅ Real-time WiFi connection status
- ✅ Latency (Ping) monitoring
- ✅ Download and upload speed estimation
- ✅ Network quality scoring system
- ✅ Automatic detection of disconnected state
- ✅ Displays "No Connection" when offline

## 🛠️ Technologies Used
- Node.js
- Express.js
- Electron (for desktop app)
- HTML, CSS, JavaScript
